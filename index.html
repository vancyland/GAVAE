Data First

From Unified Image Model to World Model

理想目标：面向用户意图与当前场景，模型可理解—规划—执行，并生成一致的关键帧与完整视频。 现实缺口：现有统一图像模型在3D空间感知与生成上存在基础能力不足，直接削弱了规划与生成的可信性。 为此我们构建一个自下而上的层级化数据集，覆盖四个关键维度，促使统一图像模型迈向世界模型。

一、数据分布：四个维度

Distribution
维度说明： 1) 物体基础集合属性感知；2) 物体-centric 空间方位感知；3) Navigation 空间感知（视点/路径与多图推理）； 4) 具身操作空间感知（理解—规划—操作—生成的序列化场景）。
维度一聚焦对象的基础属性与几何特征，提供后续方位与交互的稳定锚点。
维度二围绕“物体为中心”的相对方位、近远层级与遮挡关系，强调局部—整体的拓扑一致性。
维度三考察跨视点/路径的空间连续性与可预测性，并引入多图推理以保持结构与时序自洽。
维度四以具身操作序列为目标形态，检验理解—规划—操作—生成的闭环一致性，指向“世界级”能力。

二、系统化数据构建 Pipeline

Pipeline
数据来源与融合：以 VLM 为驱动，汇聚 ScanNet、Omniworld、AGIBot、nuScenes 等真实数据，并结合 Minetest/Libero 等仿真与游戏域数据补强长尾与可控性。
语义注入与自动筛选：注入部分 VQA 标注，引导自动化构建与筛选各维度的编辑对与对照样本，提升结构/拓扑信号密度。
质量增强（可选）：对代表性样本使用 GPT-4o 等进行语义与视觉增强，覆盖难例并提升可读性。
目标：以可扩展流程将多域样本自动组织到四个层级维度，形成递进的空间能力土壤，服务“从统一图像模型到世界模型”的路线。

三、GAVAE：为何在 VAE 上发力，以及如何注入 3D 归纳偏置

GAVAE
为何聚焦 VAE：潜空间结构决定世界建模的可分解与可控。传统2D VAE以像素重建为主，3D关系易被弱化，不利于承载世界状态。
改进方向：对齐 3D 编码器（如 CUT3R）的中层几何特征分布，将深度/法线/表面方向等几何代理注入潜空间，使其对3D结构更敏感。
对齐动机：以3D特征为教师分布，约束VAE编码器输出的分布一致性，促进潜空间承载拓扑与几何信息，支撑导航与具身操作中的连贯生成。
GAVAE并非端到端世界模型，而是提供更具3D可读性的潜空间基座，与本数据集的四维度递进相配合，缩短“统一图像模型→世界模型”的距离。
